Soundsplainer README

What this is: Soundsplainer is a browser-native Web Audio palette board built for user-to-LLM communication. It lets you audition simple generated sounds, tune them with sliders, and copy reusable sound codes.

Why it exists: describing a sound with words is vague. A copied code like POPP-08 V28:L180:G105:F35 gives a future dev session an exact reference for the selected sound family, pad number, and slider settings. The KEY button copies the decoder block you should paste into a new LLM chat so the model knows what categories like POPP, CHIR, or TICK mean.

  • VOL / V: playback volume.
  • LEN / L: sound length in milliseconds.
  • GAP / G: repeat gap used when loop is on.
  • FX / F: effect intensity, interpreted differently by sound type.

Workflow: click pads until you hear a useful sound, adjust sliders, use LOOP if needed, then COPY the final code. If you are in a fresh LLM session, click KEY and paste that decoder first, then paste your copied sound code. This gives the model both the code and the meaning behind the sound families.

Controls: LOOP repeats the last/next selected pad, STOP kills current audio and loop, COPY copies the last full sound code, KEY copies the Soundsplainer decoder/key, README explains the tool, ESC stops, L toggles loop, Enter copies.
